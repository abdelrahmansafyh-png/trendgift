# Product price visibility

تمت إضافة خيار من الأدمن لإظهار/إخفاء أسعار المنتجات.

المكان:

```txt
/admin/settings
```

الخيار:

```txt
إظهار أسعار المنتجات في الموقع
```

إذا ألغيت التفعيل، تختفي الأسعار من:

- كروت المنتجات في الصفحة الرئيسية
- صفحة تفاصيل المنتج
- رسالة واتساب الخاصة بصفحة المنتج

مهم: شغّل ملف SQL التالي في Supabase SQL Editor مرة واحدة:

```txt
supabase/show_product_prices_setting.sql
```
