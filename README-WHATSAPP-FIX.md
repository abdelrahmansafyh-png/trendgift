# WhatsApp number behavior

تم توحيد كل أزرار واتساب على رقم صفحة تواصل معنا من الأدمن.

- الرقم الوحيد الأساسي يتم تعديله من: `/admin/contact`
- صفحات Story تستخدم نفس الرقم تلقائيًا عند اختيار زر واتساب.
- لا يوجد رقم واتساب منفصل داخل Story Builder.
- حقل رابط الزر يظهر فقط عندما يكون Action = رابط.
